# AI Resume Analyzer

An ATS Resume Screening Application built using Python, Streamlit, NLP and Machine Learning concepts.

## Features

- Resume PDF Parsing
- Skill Extraction
- ATS Score Calculation
- NLP Similarity Matching
- Missing Skill Detection
- Resume Strength Analysis
- ATS Report Download

## Tech Stack

- Python
- Streamlit
- Pandas
- Plotly
- Scikit-Learn
- NLP
